# Example: 人生要做的 20 件事

这个示例来自阿甜 NewLife 内容库，用来展示 `knowledge-card-journal` 适合处理的素材类型：

- 小红书式图文内容
- 高信息密度手帐图
- 目标管理 / 能量管理类知识卡
- 可以进一步拆成卡片、海报、公众号的素材

## Source Summary

主题：AI 做图规划：人生要做的 20 件事

核心内容：

- 固定起床时间
- 晨起一杯温水
- 晨间离线 30 分钟
- 整理床铺
- 规律拉伸 / 运动
- 吃掉那只青蛙
- 番茄工作法
- 手机物理隔离
- 三项核心清单
- 工作台面清零
- 每日阅读
- 定期数字排毒
- 拒绝低价值社交
- 反思式复盘
- 控制输入源
- 预备明天
- 睡前电子禁令
- 固定记账 / 复盘支出
- 定期物理断舍离
- 冥想 / 深度呼吸

## How This Skill Would Handle It

这个素材不适合直接做成长文总结，更适合两种输出：

1. 拆成 4 张知识卡：能量启动、生产力爆发、认知心态、系统维护。
2. 整理成一张 3:4 高信息密度手帐海报。

## Sample Output Direction

### 主题

从 20 个小动作开始搭建自己的能量管理系统

### 核心判断

真正有用的习惯规划，不是一次性列很多愿望，而是把一天拆成可重复的小系统：早晨启动、白天专注、心态整理、晚上维护。

### 卡片 1

标题：早晨先把能量点亮

- 固定起床时间
- 晨起喝水
- 起床后先离线 30 分钟
- 整理床铺，让一天有一个小秩序

### 卡片 2

标题：白天只抓最重要的事

- 先做最难任务
- 用番茄钟保护专注
- 手机放远一点
- 每天只列 3 个核心任务

### 卡片 3

标题：输入少一点，判断清一点

- 每天读一点书
- 定期数字排毒
- 少接触低价值社交
- 用复盘替代情绪内耗

### 卡片 4

标题：晚上是在维护系统

- 提前准备明天
- 睡前减少屏幕刺激
- 记录钱流向哪里
- 用冥想和呼吸给身体收尾

### 一句判断

这组内容更适合整理成行动卡，重点不是 20 件事全做完，而是先搭出一个能持续运行的小系统。

### 下一步

可以继续改成小红书发图版、3:4 手帐海报版，或公众号展开版。
