# Poster Template

## Purpose

This template is for the dense A Tian hand-account infographic poster style.
It is not a card carousel.
It is one complete 3:4 poster that teaches a topic on one page.

## Best-fit topics

Works best for:
- 方法类
- 清单类
- 结构分析类
- 时间管理类
- 学习方法类
- 成长系统类

Not ideal for:
- very emotional writing
- abstract essays with no structure
- highly technical dense detail

## Fixed layout

### A. Top title area

Use:
- 1 strong main title
- 1 short subtitle
- 3-4 tags

Example:
- 主标题：你的通勤时间是多少？
- 副标题：别把每天 2 小时，白白刷掉了
- 标签：通勤成长 / 时间管理 / 碎片利用 / 普通人版本

### B. Intro judgment area

Goal:
- quickly explain why the topic matters
- give one number, contrast, or hook

Good shapes:
- `一年 730 小时，可以读完 30-50 本书`
- `以前通勤是刷手机，现在通勤是一段低成本成长时间`

### C. Main teaching blocks

Default is 4 blocks.

Each block should contain:
- a short title
- 2-4 bullets
- 1 highlighted takeaway

Useful block patterns:
- 阶段 1 / 阶段 2 / 阶段 3 / 阶段 4
- 如果是 A / 如果是 B
- 方法一 / 方法二 / 方法三 / 方法四
- 问题 / 误区 / 做法 / 总结

### D. Character and icons

Keep a central mascot when possible.

Rules:
- mascot should support friendliness and continuity
- icons should match the content block
- decorative icons should be sparse but frequent enough to keep the poster alive

Good icon types:
- 时钟
- 书本
- 清单
- 大脑
- 播客耳机
- 地铁 / 汽车
- 箭头 / 星星 / 胶带

### E. Bottom summary

Use:
- one small summary title
- 2-3 short wrap-up lines
- one memorable closing sentence

Good example:
- `哪怕只拿出每天 1 小时，一年也是 365 小时。`
- `普通人不需要做到极致，也可以抓住碎片时间。`

## Compression rules

When turning long content into poster content:

1. Keep only one core claim.
2. Merge similar examples.
3. Cut any sentence that cannot fit one visual block.
4. Replace explanation with labels, bullets, and highlighted takeaways.
5. If a section needs more than 4 bullets, split it or reduce it.

## Example topic routing

### Topic: 通勤时间

Good poster split:
- 引言：730 小时到底有多大
- 模块 1：爱凌的启发
- 模块 2：开车 / 打车怎么用
- 模块 3：地铁 / 公交怎么用
- 模块 4：早晚高峰分开用
- 总结：不要把通勤变成第二个办公室
