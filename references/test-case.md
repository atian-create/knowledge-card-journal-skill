# Test Case

## Example Input

标题：为什么自由职业的人总觉得自己很忙但没结果

正文大意：
- 每天做很多杂事
- 一会儿学这个一会儿学那个
- 看起来很努力，但没有持续积累
- 真正的问题不是方向太多，而是精力太散

## Example Output

### 主题
自由职业者为什么总觉得自己很忙但没结果

### 核心判断
很多时候问题不是不努力，而是时间和注意力被切得太碎，最后什么都碰了一点，却没有一个东西真正长出来。

### 卡片 1
标题：你不是做得少，是太散

- 真正的问题通常不是不努力
- 而是精力被切成了很多小块
- 每件事都在推进一点，但都没推进深

### 卡片 2
标题：杂事会制造努力的幻觉

- 回消息、改细节、查资料
- 都会让人觉得今天很充实
- 但它们不一定形成真正积累

### 卡片 3
标题：自由职业最怕没有主线

- 没有主线时，什么都像重要
- 每天都在处理事情
- 但没有一个结果被真正做出来

### 卡片 4
标题：解决方法不是更拼，是收束

- 只留一个主任务
- 让其他动作都服务这条主线
- 先把一颗子弹打穿，再谈扩张
